# Sleekspace Premium

A polished Next.js landing page for a premium interior design or creative agency website.

Built with modern frontend tooling and structured around reusable React components, this project showcases a full one-page layout with hero, portfolio, services, testimonials, and contact sections.

## Features

- Responsive single-page layout
- Hero section with attention-grabbing branding
- Projects / portfolio showcase
- About, Founder, Services, Process, and Why Choose Us sections
- Testimonials carousel powered by Swiper
- Sticky navigation and footer
- Tailwind CSS styling with a clean, minimal UI

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Swiper
- Lucide React icons

## Project Structure

- `app/` - Next.js application entry and routing
- `app/layout.js` - Global layout and metadata
- `app/page.js` - Main homepage composition
- `components/` - Reusable UI sections and components
- `components/testimonials/` - Testimonials carousel component
- `public/` - Static assets and public files
- `globals.css` - Global Tailwind and base styles

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

3. Open the app in your browser

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
npm run start
```

## Customize

- Update the page title and metadata in `app/layout.js`
- Adjust styling in `app/globals.css`
- Modify section content in `components/*`

## Notes

This project is configured as a private Next.js application and is ready to be extended for a custom interior design, creative agency, or product landing page.
